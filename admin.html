GitHub Token einrichten

Damit Quests für alle sichtbar gespeichert werden, wird ein GitHub Personal Access Token benötigt. Dieser wird nur lokal in deinem Browser gespeichert.

1. Gehe zu github.com/settings/tokens/new
2. Gib einen Namen ein (z.B. shard-preise)
3. Setze Ablaufdatum auf No expiration
4. Aktiviere nur: repo (alle repo-Rechte)
5. Klicke "Generate token" und kopiere den Token
